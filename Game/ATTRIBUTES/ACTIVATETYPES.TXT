/************************************************************************
**
**                          --[ACTIVATETYPES]--
**
** ActivateTypes are a property in DEdit that correspond to either linear
** or binary actions in TRON 2.0. The property is associated with objects
** that the player can directly interact with in the game.
**
** The following objects are currently known to use this attribute:
** BinaryBit,
**
** Because some classes in DEdit have static, unchangeable references to
** an ActivateType, it is not recommended to modify or remove any of the
** existing entries, and instead add your own to the end of the list.
**
** All ActivateTypes definitions should contain the following properties:
**
**  Name (String)
**
**	The user friendly name of the ActivateType (what shows up in DEdit).
**
**  StateON (integer)
**
**	A string ID from CRES.DLL of the text to display whe the object is in
**	it's ON state.  Use -1 for no text.
**
**  StateOFF (integer)
**
**	A string ID from CRES.DLL of the text to display whe the object is in
**	it's OFF state.  Use -1 for no text.
*/
