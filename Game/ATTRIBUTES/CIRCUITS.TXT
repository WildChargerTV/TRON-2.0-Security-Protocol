/*

	CIRCUITS

NameKey				(String)
	Name used to refer to this circuit for unlocking, etc.

Image				(String)
	Name of the Image to be displayed on the circuit page

CircuitNameId		(Integer)
	Id of the circuit to display.

BriefingId		(Integer)
	Id of the string describing this circuit

StandardCycleRestrictionId	(Integer)
	Id of string describing which standard cycles are allowed in this circuit

SuperCycleRestrictionId	(Integer)
	Id of string describing which super cycles are allowed in this circuit

CircuitCompleteId (Integer)
	Id of string expressing congratulations for the circuit completed.

CircuitCompleteSound (String)
	Filename of the sound to play upon completion of a circuit.

MapName					(String)
	Map name for the circuit

InitiallyLocked		(TRUE/FALSE)
	Does this mission start out locked.

Unlocks0-N		(String)
	List of contents to be unlocked when this circuit is beat.
	Valid entries:
		Powerups (one of these entries):
			Shield
			Turbo
			TrailSpike
			Missile
			WallExtender
			TurboCurse
			Grab
			TrailReset
			ForceUse
			Powerup
			Random

InitialLives			(Integer)
	Lives given to the player to attempt this circuit on starting circuit.


Per Wave Settings:

SpeedScalarId0-N			(Integer)
	Speed for this wave.  Valid values are 0 (Slow), 1 (Normal), 2 (Fast), 3 (Extreme)

TrailLengthId0-N		(Integer)
	Trail Length for this wave.  Valid values are 0 (Short), 1 (Normal), 2 (Long), 3 (Extreme)

CyclesAllowen0-N		(String)
	Names of all the light cycles the player is allowed to play in this circuit.

*/

Current ids for a few of the restrictions:

red		- 9723
green	- 9724
purple	- 9728
*/
